Nombre: GamesPortal
Autor: David González
Versión: 0.1.0

DESCRIPCIÓN
------------------------------------------------------------------------------------------------
GamesPortal es un sencillo gestor de contenidos desarrollado con PHP, HTML, CSS, JS y SQL que
permite dar vida a una aplicación Web sobre videojuegos. Con esta herramienta redactar entradas
(análisis, noticias, rumores, avances, guías y artículos), mantener fichas de videojuegos,
gestionar un foro, mensajería privada y mucho más.


CARACTERÍSTICAS DESTADAS
------------------------------------------------------------------------------------------------
	- Panel de administración
	- Panel de usuario
	- Roles de usuario
	- Páginas securizadas
	- Entradas
	- Comentarios
	- Foro
	- Mensajes privados
	- Plataformas
	- Fichas de videojuegos
	- Guías
	- Trucos
	- Logros
	- Sistema de reportes
	- Valorar videojuegos
	- Top de videojuegos
	- Sistema de plantillas (para personalizar el aspecto de tu sitio)


IMPORTANTE
------------------------------------------------------------------------------------------------
Este proyecto aún se encuentra en proceso de desarrollo y está siendo codificado por simple
entretenimiento.
